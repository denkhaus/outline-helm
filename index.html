Outline Helm Chart Repository

Enterprise-grade Helm chart for deploying Outline wiki in Kubernetes

🚀 Quick Start

Add the repository:
helm repo add outline-helm https://denkhaus.github.io/outline-helm

Update repositories:
helm repo update

Install Outline:
helm install outline outline-helm/outline

📋 Prerequisites

Before installing, create a Kubernetes secret with required values:

kubectl create secret generic outline-secrets \
  --from-literal=SECRET_KEY="$(openssl rand -hex 32)" \
  --from-literal=UTILS_SECRET="$(openssl rand -hex 32)" \
  --from-literal=SLACK_CLIENT_SECRET="your-slack-secret" \
  --from-literal=SMTP_PASSWORD="your-smtp-password"

✨ Features

  • 🛡️ Security First: All sensitive values via Kubernetes secrets
  • 📦 Complete Stack: Outline + PostgreSQL + Redis + MinIO
  • 🔧 Flexible: Optional ingress, configurable dependencies
  • 🚀 Production Ready: Enterprise-grade security and reliability
  • 🤖 Automated: Dependabot updates and comprehensive testing
